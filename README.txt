Revit Version : 2022
Revit Launchers: pyRevit, etc.

Description: The program collects levels from all open linked files in the model. Compiles a string and writes this string to the clipboard for pasting into a spreadsheet.
Note: The program does not take into account the levels in the document itself.

Описание: Программа собирает уровни со всех открытых связанных файлах в модели. Составляет строку и записывает эту строку в буфер обмена для вставки в электронную таблицу.
Примечание: Программа не учитывает уровни в самом документе.
